2.4.3 部署更新

每次更新本地文件后，运行以下命令：

  1. hexo clean
  2. hexo generate
  3. hexo deploy

至此，初始化的网站已经配置完毕，您可以通过 GitHub Pages 的地址访问。

小贴士：

  • 每次配置后都需运行以上三步。
